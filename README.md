# DashboardXBoxDio

Este repositório contém o desenvolvimento de um Dashboard de Vendas focado em vendas de produtos Xbox. 
O objetivo foi transformar dados brutos de transações em uma interface visual que permite qualquer um analisar o desempenho comercial de forma rápida e precisa.

Desafio
O projeto consistiu em processar uma base de dados de vendas e construir um painel interativo que respondesse às principais perguntas de negócio:

Pergunta 1 - Qual faturamento Total de vendas de planos anuais (contendo todas as assinaturas agregadas)
Pergunta 2 - Qual Faturamento Total de vendas de planos anuais , separado por auto renovação não é por auto renovação
Pergunta 3 - Total de Vendas de Assinaturas do EA Play
Pergunta 4 - Total de Vendas de Assinaturas do Minecraft Season Pass

Estrutura do Arquivo
Para garantir a organização e escalabilidade (boas práticas de BI), o arquivo foi estruturado em camadas:

B̳ases: Dados brutos de vendas.

C̳álculos: Camada de processamento com fórmulas e tabelas dinâmicas que alimentam os visuais.

A̳ssets: Elementos visuais e ícones utilizados para compor a identidade visual do Xbox.

D̳ashboard: Interface final interativa para o usuário final.

Competências Aplicadas
Tratamento de Dados: Limpeza e organização de bases de dados reais.

Fórmulas: Uso de lógica para consolidação de métricas.

Design de Dashboards: Criação de layout intuitivo e profissional focado em experiência do usuário (UX).


Como Reproduzir
Faça o download do arquivo dashboardXBoxDio.xlsx

Abra no Microsoft Excel.

Navegue pela aba Dashboard para interagir com os dados.

Consulte a aba Cálculos para entender a lógica de processamento por trás dos gráficos.
